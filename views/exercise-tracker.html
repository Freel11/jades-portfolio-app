Exercise tracker

For this project I had to connect to a database and create a collection for both users and exercises. the users have to be unique (cannot use a taken username) and the exercise has to relate to a specific user.

You can add a new user through the form on the left and add an exercise through the form on the right. You can then query the exercises logged by a specific user and filter the results that you receive.

Create a New User

POST /api/users

Add exercises

POST /api/users/:_id/exercises

GET user's exercise log: GET /api/users/:_id/logs?[from][&to][&limit]

[ ] = optional

from, to = dates (yyyy-mm-dd); limit = number
